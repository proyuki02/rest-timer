# Rest Timer

![Rest Timer](https://user-images.githubusercontent.com/40527123/184806719-8e575180-9627-418a-b671-7c26984e422c.png)

## 概要
昼休み、および定時になったらお知らせするアプリです。

## 仕様
* 終了までの残り時間を表示します。
* 終了3分前になると最前面に表示されます。
* 終了すると全画面表示されます。
* 昼休みになったらタスクトレイのアイコンをダブルクリックすると、定時のお知らせに切り替わります。

## 使用するコマンド
```
npm install
npm start
npm run package
```
